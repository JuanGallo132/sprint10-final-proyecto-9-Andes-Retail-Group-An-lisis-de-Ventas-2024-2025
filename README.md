# 📊 Andes Retail Group — Análisis de Ventas 2024–2025

## 📌 Descripción del proyecto

Este proyecto presenta un análisis exploratorio de ventas y rentabilidad para Andes Retail Group utilizando Power BI. El objetivo principal es identificar patrones en el comportamiento del negocio, evaluar el desempeño por segmentos de clientes, categorías de productos y mercados geográficos, y detectar oportunidades de mejora en ingresos y margen.

A través de dashboards interactivos se analizan indicadores clave como ingresos, ganancia bruta, volumen de ventas y pedidos procesados, permitiendo comprender la evolución del negocio entre 2024 y 2025.

## 🎯 Objetivos del análisis

* Analizar la evolución de ingresos y margen en el periodo 2024–2025.
* Identificar patrones de estacionalidad en las ventas.
* Evaluar el desempeño por segmento de cliente.
* Analizar la distribución de ingresos por categoría de producto.
* Comparar resultados entre mercados geográficos.
* Proponer insights estratégicos para mejorar el rendimiento del negocio.

## 📈 Principales hallazgos

* Los ingresos totales alcanzan 2,67M, mostrando una disminución aproximada del 6,8% respecto a 2024.
* Se identifica una estacionalidad clara entre junio y agosto, donde las ventas presentan una caída recurrente.
* Los segmentos Premium y Estándar generan cerca del 90% del ingreso total, siendo los principales impulsores del negocio.
* Las categorías de Deportes, Electrónica, Hogar y Ropa presentan ingresos similares, indicando un portafolio equilibrado.
* Perú es el mercado más fuerte, representando aproximadamente 39% del ingreso total.
